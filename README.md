# 🔐 Password Generator Tool (Python)

A simple and secure Password Generator Tool built using Python. This application allows users to generate strong, random passwords with customizable length and character combinations.


## 🚀 Features

* Generate secure random passwords
* Includes:

  * Uppercase & lowercase letters
  * Numbers
  * Special characters
* User-defined password length
* Input validation and error handling
* Simple GUI built with Tkinter

## 🛠️ Technologies Used

* Python
* Tkinter (for GUI)
* Random module
* String module

## 📸 Output

* Enter desired password length
* Click "Generate Password"
* Secure password will be displayed instantly
